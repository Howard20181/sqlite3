# SQLite3 for arm, arm64, x86 and x64

## Description

This Magisk module provides a statically linked and stripped `sqlite3` binary for **arm64-v8a**, **armeabi-v7a**, **x86** and **x86_64** devices.

## Tested Devices

 - Lenovo Tab M10 Plus Gen 3
 - LDPlayer 9
